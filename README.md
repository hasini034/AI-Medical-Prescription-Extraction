# 💊 AI Medical Prescription Extraction

## 📌 Project Overview

AI Medical Prescription Extraction is a Streamlit web application that extracts structured information from a medical prescription.

The application identifies:
- 👤 Patient Information
- 👨‍⚕️ Doctor Information
- 💊 Medicines
- 📅 Prescription Date
- 📌 General Instructions
- 
## 🚀 Features

- Extract Patient Details
- Extract Doctor Details
- Extract Medicine Details
- Extract Prescription Date
- Display data in JSON format
- Display medicines in a table
- Simple and user-friendly interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Regular Expressions (Regex)

## ▶️ Installation

Clone the repository

 bash

 bash

 bash
